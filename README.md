# 🐍 霓虹贪吃蛇 (Neon Snake)

一款具有精美霓虹视觉效果的网页版贪吃蛇游戏，采用纯 HTML5 + CSS3 + JavaScript 开发。

![游戏预览](https://img.shields.io/badge/版本-1.0.0-cyan)
![许可证](https://img.shields.io/badge/许可证-MIT-green)

## ✨ 特性

### 🎮 游戏功能
- **经典贪吃蛇玩法** - 控制蛇吃食物，不断成长
- **用户系统** - 支持注册、登录、游客模式
- **数据记录** - 自动保存最高分、游戏次数、总得分
- **渐进难度** - 分数越高，速度越快

### 🎨 视觉效果
- **深色霓虹主题** - 深海蓝背景 (#0f172a)
- **动态背景** - Cyan 和 Fuchsia 色发光球体缓慢漂浮
- **毛玻璃效果** - backdrop-filter 模糊玻璃面板
- **霓虹发光** - 按钮和元素带有 box-shadow 发光特效
- **渐变标题** - Cyan → Fuchsia 线性渐变文字

### 🐍 卡通角色
- **卡通蛇头** - 带眼睛和舌头，会根据移动方向转向
- **丰富食物** - 随机出现苹果 🍎、老鼠 🐭、青蛙 🐸

### 💫 交互体验
- **响应式设计** - 适配桌面和移动设备
- **键盘控制** - 方向键或 WASD 控制
- **微动效** - 按钮悬停发光增强
- **现代字体** - Outfit 字体，科技感十足

## 🚀 快速开始

### 方式一：直接打开
双击 `index.html` 文件在浏览器中打开即可游玩。

### 方式二：本地服务器（推荐）
```bash
# 进入项目目录
cd snake-game

# 使用 Python 启动本地服务器
python -m http.server 8080

# 或使用 Node.js
npx serve .
```
然后访问 http://localhost:8080

## 🎯 游戏操作

| 按键 | 功能 |
|------|------|
| `↑` `W` | 向上移动 |
| `↓` `S` | 向下移动 |
| `←` `A` | 向左移动 |
| `→` `D` | 向右移动 |
| `空格` | 开始 / 暂停 |

## 👤 用户系统

### 注册账号
- 用户名：3-12 个字符（支持中文、字母、数字、下划线）
- 密码：至少 6 个字符

### 游客模式
- 无需注册，点击"游客模式"即可体验
- 游客数据不会保存

### 数据存储
- 使用浏览器 LocalStorage 本地存储
- 数据保存在用户浏览器中，无需服务器

## 🛠️ 技术栈

- **HTML5** - 语义化页面结构
- **CSS3** - 动画、渐变、毛玻璃效果
- **JavaScript (ES6+)** - 游戏逻辑和交互
- **Canvas API** - 游戏画面渲染
- **LocalStorage** - 本地数据持久化

## 📊 数据统计

游戏会自动记录以下数据（仅限注册用户）：
- 🏆 最高分数
- 🎮 游戏次数
- 📈 累计总分
- ⏰ 登录时间
